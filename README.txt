What it does
- Upload video, image, and audio files locally.
- Assign them to stacked timeline layers.
- Toggle each layer visible/hidden.
- Drag clips in time and between layers.
- Resize clip duration from the left or right edge.
- Preview composited visible image/video layers.
- Play synced audio/video along the timeline.
- Save and load projects as .layercut files.
- Export the full timeline as WebM.
- Add text clips over picture/video.
- Add privacy scrub regions to image/video clips.
- Add visual privacy scrub regions to selected image/video clips.

Privacy tools
1. Select an image or video clip.
2. Use Draw Region to draw a blur, pixelate, or black-bar region on the preview.
3. Use the Security Scrub controls to draw privacy regions on the selected image/video clip. These regions are burned into preview and WebM export.

Notes
- Privacy scrub regions are burned into preview and export.
- Videos are re-recorded in-browser as WebM, which strips typical source container metadata but depends on browser support.
- Browser autoplay rules may require a click on Play before media audio is allowed.
- Some video codecs may not decode in every browser.
